老版本的IDEA在左侧的Project tab页下面都有一个Commit的对话框，但是在2022版本里默认不显示。

<!--more-->

## 打开方式

打开设置，依次找到 Settings -> Version Control -> Commit，勾选`Use non-modal commit interface`。

![img_1.png](/img/ide/idea/2023-05-28/img_1.png)

此时，在左侧就能找到 Commit 对话框了。

还是熟悉的配方 😎
